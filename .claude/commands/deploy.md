Vercelへデプロイ（ユーザー確認必須）

手順:

1. **プリフライトチェック**
   - `npx tsc --noEmit` — 型エラーゼロ確認
   - `npm run lint` — lint通過確認
   - `npm run build` — ビルド成功確認

2. **チェック結果報告**
   - エラーがあれば修正提案して停止
   - 全パス → ユーザーにデプロイ許可を求める

3. **デプロイ実行**（許可後のみ）
   - `npx vercel --prod`
   - デプロイURL報告

4. **デプロイ後検証**
   - Playwright MCPで本番URLにアクセス
   - 3Dシーンのレンダリング確認
