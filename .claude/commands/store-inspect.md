Zustandストアの指定セクションを安全に読む: $ARGUMENTS

useEditorStore.ts は ~2000行のモノリス。全読みはコンテキスト爆発。
このコマンドで必要部分のみ読む。

手順:

1. **セクション特定**
   - 引数にキーワード指定あり → Grep でそのキーワードの行番号を特定
   - 引数なし → ストアの構造概要（interface定義部分のみ）を表示

2. **ピンポイント読み込み**
   - 特定した行番号 ± 30行のみを Read (limit パラメータ使用)
   - 関連する型定義があれば `src/types/` から該当部分のみ読む

3. **報告**
   - 読んだセクションの概要を簡潔に報告
   - 関連するアクション/セレクタの一覧

注意: ストア全体を読まない。必要なセクションだけ。
