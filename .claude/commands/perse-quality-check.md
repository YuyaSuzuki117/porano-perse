パースレンダリング結果の品質チェックを実施する。

## 引数
- `$ARGUMENTS`: チェック対象の画像パス

## 手順

### 1. レンダリング画像を表示
- Readツールで画像を表示
- 参照画像がある場合は並べて確認

### 2. 建築パース品質チェックリスト

#### A. 構造 (最重要)
- [ ] 壁の垂直性: 壁が垂直に立っているか
- [ ] 天井高: 自然な高さに見えるか
- [ ] 床の水平性: 床が水平か
- [ ] 壁と天井の接合: クリーンなエッジか
- [ ] 壁と床の接合: 巾木が適切か
- [ ] 開口部: ドア・窓の位置とサイズが正確か

#### B. マテリアル
- [ ] 床材: テクスチャの方向・スケールは適切か
- [ ] 壁仕上げ: 色味・質感は指定通りか
- [ ] 天井: 白飛びや暗すぎはないか
- [ ] 金属部品: 反射が自然か
- [ ] ガラス: 透過・反射のバランスは適切か

#### C. ライティング
- [ ] 全体の明るさ: 自然な明るさか
- [ ] 影の方向: 光源と整合しているか
- [ ] コントラスト: 暗すぎ・明るすぎの部分はないか
- [ ] 色温度: 空間の雰囲気に合っているか

#### D. カメラ
- [ ] アングル: 空間を効果的に見せているか
- [ ] 歪み: パース歪みが不自然でないか
- [ ] 構図: 主要な要素が収まっているか

#### E. 禁止事項チェック
- [ ] 浮遊オブジェクトがない
- [ ] 壁を貫通しているオブジェクトがない
- [ ] 極端なノイズ・アーティファクトがない
- [ ] 黒い面（法線反転）がない

### 3. 採点（100点満点）
| カテゴリ | 配点 | 評価基準 |
|---------|------|----------|
| 構造正確性 | 40点 | 寸法・比率・接合 |
| マテリアル | 20点 | 質感・色味 |
| ライティング | 20点 | 明るさ・影・雰囲気 |
| カメラ/構図 | 10点 | アングル・歪み |
| クリーンさ | 10点 | エラー・アーティファクト |

### 4. 改善提案
- スコア別の具体的な改善点を3つ以内でリスト
- 各改善点について修正の難易度（低/中/高）を付記
- `/perse-iterate` で改善する際のコマンド引数を提示
