Playwrightで3Dシーンのビジュアルテスト: $ARGUMENTS

手順:

1. dev serverが起動していなければ `npm run dev` をバックグラウンド実行
2. Playwright MCPでブラウザを開き `http://localhost:3001` にアクセス
3. 3Dシーンの読み込み完了を待機（Canvasレンダリング完了）
4. 以下のテストを実行:
   - スクリーンショット取得（全体ビュー）
   - スタイル切替（各スタイルのスクリーンショット）
   - 昼/夜モード切替の確認
   - 家具配置の確認
   - レスポンシブ（モバイル幅でのレンダリング）
5. 問題があれば報告、なければ「正常」と報告

指定があればそのスタイル/機能のみテスト。
