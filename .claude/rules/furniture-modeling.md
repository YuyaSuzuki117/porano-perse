# Furniture Modeling

## 家具全般

- 参照画像が複数あるなら、正面よりも「完成イメージに近い主視点」を優先してシルエットを合わせる。
- 設計図と写真が矛盾する場合は、先に写真の見え方を満たし、その後で寸法を寄せる。
- 接続部が曖昧な時は、浮いて見えない方を採用する。

## テーブル

- 丸テーブルの天板は、厚み、エッジの丸み、支柱径の 3 点で印象が決まる。
- ペデスタルテーブルは、`天板直径 / 支柱径 / ベース直径 / 全高` の比率を先に合わせる。
- ペデスタル下部は、`円柱 + コーン + 円盤` の分割で境目が出やすい。
- 参照で一体成形に見える場合は、回転体プロファイルを使って 1 メッシュで作る。
- ベース底面が薄い円盤に見えても、真横から見た時の持ち上がり方が重要。フラットすぎると安っぽく見える。

## 椅子

- 背もたれ、後脚、座面フレームの接続は絶対に嘘をつかない。
- 曲木フレームはカーブで作り、断面は bevel object か bevel depth で与える。
- 座面より脚の開きが先に狂うことが多い。脚位置は早めに検証する。

## 形状選択の目安

- 円形天板: cylinder
- テーパー脚: cone / custom profile
- 曲線フレーム: curve
- 一体ペデスタル: lathe profile
- 単純な板材: cube か plane + solidify

## よくある失敗

- 支柱が太すぎて軽さが消える
- ベースが平板すぎて別部品の貼り合わせに見える
- 背もたれや脚の接続が写真よりも外側に逃げる
- 形が合っていないのに材質で誤魔化そうとする
